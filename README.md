## Description
This repository made for build simple of RabbitMQ with docker.

## Prerequisite
* [Docker](https://docs.docker.com/engine/install/ubuntu/)
* [Docker Compose](https://docs.docker.com/compose/install/)

## Run
```bash
    docker-compose up -d
```
